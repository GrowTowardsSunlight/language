非顺序的程序控制，往往需要根据一定的条件来决定程序运行的路线。

### 比较运算
```python
a, b = 10, 8
a > b   #大于
a < b	#小于	
a >= b	#大于等于	 
a <= b	#小于等于
a == b	#等于
a != b  #不等于
```
- 非空
```python
ls = []
if ls:		#数据结构不为空，变量不为0、None、False，则条件成立。
	print('非空')
else:
	print('空的')
```

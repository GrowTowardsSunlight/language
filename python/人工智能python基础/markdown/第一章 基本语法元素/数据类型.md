**基本数据类型**

数字类型

	整型int、浮点型float、 数complex
字符串类型

	str用''或者""表达
布尔类型
	
	bool可以表示逻辑的是与非

**组合数据类型**

列表

	list有序表,列表中的元素可以被修改。
	表示方式：[data1,data2,...datan]
	 
元组

	tuple有序表,元组中的元素不可以被修改。
	表示方式：(data1,data2,...datan)
	
字典
	
	dict映射类型，无序表，通过“键-值”的映射实现数据存储和查找
	表示方式：{key1:value1,key2:value2,...}
集合

	set无序表，元素互不相等。
	表示方式：{data1,data2...}
